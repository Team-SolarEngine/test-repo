# TEST-REPO
This repo is a test for a new feature being added where it would read [.solar-engine](.solar-engine) folder for [banner.png](.solar-engine/banner.png), [config.json](.solar-engine/config.json), [logo.png](.solar-engine/logo.png) and [readme.md](.solar-engine/readme.md).

## Configuration? ([config.json](.solar-engine/config.json))
- `isOpenSource` > If it's open source, the tag will show if it is or not. (yes/no)
- `canMessWithComputer` > If it messes with your computer, it will show if it is or not. (yes/no)
>
- `title` > What the title should be for your project. (eg: **Solar Engine**)
- `description` > What the description should be for your project. Shown in the main page and not the popup.
- `madeBy` > Who made it? A team of people? or a single person? (eg: **Solar Engine Team** or **Daveberry Cricket Blueson**)
>
- `downloadURL` > What your Download URL will redirect to. You can put nothing in it if you don't have a link for it.
- `externalURL` > What your external URL will redirect to. You can put nothing in it if you don't have a link for it.
- `madeByURL` > The team/person URL who made the project. Linked to their website.
