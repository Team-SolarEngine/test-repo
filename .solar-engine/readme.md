In [the config file](config.json), please make `type` and `whatType` a one word. For example:

**Example 1:**
```json
{
  "type": "Mods",
  "whatType": "Friday Night Funkin'"
}
```

**Example 2:**
```json
{
  "type": "Website",
  "whatType": "Sharing Games"
}
```

**Example 3:**
```json
{
  "type": "Software",
  "whatType": "Utility Software"
}
```

## About the NSFW:
I was generally going to make the website ignore NSFW fully, but after some thinking and talking with a friend, I've ultimately decided to make a toggle for NSFW content.

By default, it's off so your eyes don't get blasted by NSFW content. You can turn it on in the settings if you want to see NSFW content.

For NSFW means, I generally don't mean things that are naughty. I just mean as in the thing is violent and such. I'll rethink about this soon for things that are actually NSFW (such as showing the suggestive things).
